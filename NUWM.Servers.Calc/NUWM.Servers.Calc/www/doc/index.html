Опис

  • Уточнення формату чи кількості
  • Основні - важливі
  • Пошукові
  • Застарілі

last

int [1 > ∞ ]

Повертає будь-яку кількість новин починаючи з найсвіжішої

before

[int(count)],string(url)

Повертає новини перед заданою, тобто старіші статті
1. Кількість новин перед заданою. Необов'язковий. За зам. 10
2. url новини. Обов'язковий Може бути частиною посилання на новину.

after

[int(count)],string(url),[bool(strict)]

Повертає список новин після заданої.
1. Кількість новин після заданої, тобто новіші статті
За зам. ∞.
Необов'язковий

2. url новини. Обов'язковий. Може бути частиною посилання на новину.
3. Визначає строгий запит.
За зам. [false].
Необов'язковий.
Якщо true та кількість новин менша за задану першим параметром, повернеться код 33 (не знайдено)

offset

int(offset),[int(count)]

Повертає статті пропускаючи offset,
1. Кількість новин для пропуску. Skip() Обов'язковий

2. Кількість новин у відповіді. Take() Необов'язковий.
За зам. 10

id

int

ID університету

Необов'язковий: натомість можна звернутись за ключем API, тоді calc.nuwm.edu.ua/api/nni-akot?

inst

string

Повне посилання на новину

. InstantCache: кешується "на льоту". Збереження на диск відбувається кожні 30хв. За наступним запитом новина видається з кешу.

uri

string

пошук новини в кеші парсера за повним посиланням на новину,
де парсер це ключ api?xxx

query

string

пошук по контенту в новинах в кеші парсера,
де парсер це ключ api?xxx

uriquery

string

пошук по URL в новинах

html

bool [0 or 1]

визначає формат відповіді. 0-RAW text

group

string

Назва групи. Обов'язковий. Для викладачів використовувати name

name

string

Повний ПІБ викладача, як в ПС-Студент.Обов'язковий. Для груп використовувати group

week

int[ 0 < > 52 ]

Номер тижня від початку року. Обов'язковий, інакше використовувати sdate & edate

year

int[ current year ±1 ]

Якщо задано week визначає рік. За зам. поточний. Необов'язковий

sdate

date[ dd.MM.yyyy ]

Початок розкладу. Обов'язковий, інакше використовувати week & year

edate

date[ dd.MM.yyyy ]

Кінець розкладу. Обов'язковий, інакше використовувати week & year

type

[weeks|days]

Тип відповіді. Необов'язковий. За зам. days

hlp

empty or [string,..,..]

Використовується для видачі унікальних предметів. Пустий параметр повертає усі наявні унікальні предмети. Для видачі нових актуальних предметів слід вказати вибрані в масиві. Тобто якщо вибрано Географія, повернеться список з усіх предметів крім Географії. Поодинокий(без інших ключів)

n

[string,..,..]

Масив назв вибраних предметів. Порядок має співпадати з v ключем. Обов'язковий.

v

[int,..,..]

Масив балів ЗНО користувача вибраних предметів. Порядок має співпадати з n ключем. Обов'язковий.

vl

bool

Тут все просто. Користувач із села. Обов'язковий.

pr

int

Пріоритет заяви. Обов'язковий.

avm

double

Середній бал атестату. Обов'язковий.

query

[int,string]

Поодинокий(без інших ключів)

1. Кількість новин у відповіді. Обов'язковий

2. Пошуковий запит. Обов'язковий
Прим. За наявності кешу на сервері пошук займає 1.34 сек. В іншому випадку від 5 сек при кількості 250 новин. Оптимальна кількість 5-30 новин.
При кешованому результаті поле
"cache": true;

Приклади:

calc.nuwm.edu.ua/api/ads?last=1

Остання новина 1 сторінки оголошень

calc.nuwm.edu.ua/api/nni-akot?p_id=3&last=4

Останні 4 новини 6 сторінки інституту АКОТ

calc.nuwm.edu.ua/api/news?after=nov201712201040&html=1

Усі новини після заданої

calc.nuwm.edu.ua/api/news?after=10,http://nuwm.edu.ua/university/news/nov201712201040&html=1

10 новин після заданої

calc.nuwm.edu.ua/api/news?after=10,http://nuwm.edu.ua/university/news/nov201712201040,true&html=1

10 новин після заданої тільки якщо вони є, у разі меншої кількості повернеться пуста відповідь

calc.nuwm.edu.ua/api/news?last=1

Остання новина в стрічці

calc.nuwm.edu.ua/api/calc?n=Українська мова та література,Математика,Фізика&v=180,140,150&vl=true&pr=2&avm=12

Приклад запиту на calc

calc.nuwm.edu.ua/api/sched?name=Кушнір%20Олександр%20Олегович&sdate=01.09.2017&edate=31.12.2017&year=2017&type=weeks

Розклад викладача

calc.nuwm.edu.ua/api/sched?group=ПМ-21&sdate=6.11.2017&edate=12.11.2017&type=weeks
